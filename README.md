# グラグラおつかいタワー

一人用のミニゲーム試作です。荷物を積み上げて、崩さずに配達完了まで耐えることを目指します。

## 立ち位置

- 人狼ゲーム Android 化と並行して進める別ライン
- あき案の「一人でできるゲーム」をベースにした新規チャレンジ
- まずは 1 プレイが短く、すぐ触って判断できる試作を重視

## 現在の内容

- Next.js による最小構成
- 荷物を左右中央に置く簡易プロトタイプ
- 安定度、残り時間、スコアの表示
- スマホでも確認しやすいレイアウト

## 次の候補

1. 落下アニメーションを入れる
2. 風や段差などのギミックを 1 つ追加する
3. 効果音と成功失敗演出を加える
4. Android 化しやすい入力設計に整える
