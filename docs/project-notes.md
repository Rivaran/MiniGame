# Project Notes

## コンセプト

- タイトル: グラグラおつかいタワー
- ジャンル: 一人用カジュアルゲーム
- コア体験: 荷物を積むたびに少しずつ不安定になり、崩れる前にどこまで耐えられるかを楽しむ

## 初回 MVP

- 1 プレイ 30 秒前後
- 左、中央、右の 3 択で荷物を置く
- 安定度が限界を超えたら失敗
- 制限時間を生き残れば成功

## 判断基準

- 5 秒で遊び方がわかるか
- 失敗してももう一回押したくなるか
- 見ているだけでも少し面白いか
- 今後 Android アプリに広げやすいか

## 直近の拡張候補

- 荷物が落ちてくるアニメーション
- 風で少し流されるイベント
- おつかい先ごとのテーマ差分
- 子ども向けの見た目強化
